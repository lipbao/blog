## 如何启动
node app.js

## 本项目中的依赖
- `art-temlate`
  - 这是前端模板
- `express-art-template`
  - 这是封装的前端模板的插件
- `blueimp-md5`
  - 这是密码加密的插件
- `bootstrap`
  - 这是前端样式模板
- `express`
  - 这是`node`开启服务器的插件
- `express-session`
  - 这是服务器中保存`session`的插件
- `jquery`
  - 封装的`js`库
- `mongoose`
  - 这是数据库`mongodb`封装的插件
